ESP32 Noise Detection System

About This Project

This system uses an ESP32 microcontroller connected to LM393 sound sensors. When the classroom noise level goes above a threshold, the ESP32 triggers a buzzer and LED. At the same time, the ESP32 hosts a web server that records and displays the sound levels. Teachers can monitor noise in real-time and encourage students to maintain discipline.

Features

  • Detects sound using LM393 sensor
  • Triggers buzzer and LED when noise exceeds threshold
  • ESP32 hosts a web server to show sound levels
  • Useful for classroom monitoring and discipline

Hardware Connections

  • LM393 OUT → ESP32 GPIO 15
  • LM393 VCC → 3.3V
  • LM393 GND → GND
  • Buzzer + LED → GPIO pins (configurable)
